# 10. UI Requirements (React)

## Design System
- Typography: "Space Grotesk" (H1-H4), "Source Sans 3" (body)
- Spacing scale: 4/8/12/16/24/32/48
- Buttons: primary, secondary, ghost, destructive
- Status chips: color by SLA and severity

## Navigation
- Left nav with module grouping
- Global search with fuzzy results
- Quick actions: New Incident, New Request

## Key Screens
- Executive dashboard
- Incident list/detail
- Service catalog + request wizard
- Problem list/detail
- Change calendar + CAB approvals
- CMDB graph and CI detail
- Asset inventory and lifecycle
- Knowledge base and editor

## Accessibility
- WCAG AA
- Keyboard shortcuts
- Screen-reader friendly labels
