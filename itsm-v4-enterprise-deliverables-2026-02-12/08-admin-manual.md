# Admin Manual

## Tenant Management
- Create tenant and assign domain.
- Configure branding (logo, colors) and locale.
- Set tenant-specific SLA and OLA policies.

## Identity and Access
- Configure SSO (Azure AD, Google, LDAP, SAML).
- Enable MFA and password policies.
- Define roles and permissions per module.

## Service Catalog
- Create services and request forms.
- Define approval chains and fulfillment steps.
- Configure SLA targets per service.

## Workflow Engine
- Build workflows using drag and drop states.
- Add transition conditions and timers.
- Configure notifications and auto-assign rules.

## CMDB and Assets
- Define CI classes and relationships.
- Configure discovery integrations.
- Set lifecycle rules and ownership.

## Reporting and Audit
- Enable scheduled reports (PDF/Excel).
- Review audit logs and export evidence.
- Configure SIEM integration.

## Integrations
- Email ingestion for ticket creation.
- Microsoft 365 and monitoring tools.
- Webhook endpoints for external systems.

## Operations
- Set backup schedules and retention.
- Manage license limits and tenant quotas.
- Review system health dashboard.
